Overview

I developed a client-facing project tracking system to help both Newpages customers and staff monitor project progress in real time. Before this system, customers had to call salespeople, and then salespeople had to check with the project coordinator to get updates â€” which was time-consuming and inefficient. This system replaces that manual process by giving users a secure link to track updates directly. The interface is built with Tailwind CSS and connects to real-time data through internal APIs.

Challenges Solved:

  • Gained hands-on experience with Tailwind CSS by using it for the first time to design a clean and user-friendly user interface, enhancing usability for external clients.
  • Transitioned from manual status tracking (via calls/messages) to automated, self-serve tracking.

Key Outcomes:

  • Made work easier for staff by stopping the need to answer the same update questions over and over.
  • Gave customers real-time access to project progress through secure links.
  • Increased transparency and client satisfaction with improved communication flow.
