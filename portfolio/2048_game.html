Overview

I developed this 2048 game with an integrated Q-Learning AI agent to explore how artificial intelligence could be applied to simple game logic. I initially built the classic 4x4 version, but later expanded it to support larger grids like 8x8, 16x16, 32x32 to make the game more challenging and interesting. This project was a self-initiative to combine Python programming with AI logic while also experimenting with GUI design using Tkinter.
