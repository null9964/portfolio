Overview

Upgraded the existing weekly booking system into a full monthly calendar interface using FullCalendar.js. The new system allows users to easily view available slots across all company branches. Users can now select a preferred date, browse available services, and complete their booking by filling in personal details and the number of participants. This upgrade significantly improved usability and booking transparency for both customers and staff.

Challenges Solved:

  • Replaced limited weekly booking view with a dynamic and responsive monthly calendar interface.
  • Implemented monthly booking UI using FullCalendar.js for the first time, learning it from scratch during development.

Key Outcomes:

  • Improved user experience by upgrading from a weekly to a monthly calendar view, allowing users to see all bookings in one glance without navigating week by week.
